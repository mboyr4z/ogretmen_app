## Özellikler

- Çoktan seçmeli, klasik ve doğru/yanlış soru tipleriyle sınav oluşturma
- PDF, Word ve Excel formatında sınav çıktısı
- Erişim kodlu online sınav sistemi ve gerçek zamanlı sonuç takibi
- Google OAuth2 ile kimlik doğrulama
- Ücretsiz / Premium üyelik sistemi

---

## Teknolojiler

- **Backend:** PHP, MySQL/PDO
- **Frontend:** HTML, CSS, JavaScript
- **Auth:** Google OAuth2, Session
- **Export:** TCPDF, PHPWord, SimpleXlsx
